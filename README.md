# ICA — Infinite Calculative Adventure

> The fastest arbitrary-precision calculator for everyone. In CMD!

No limits. No rounding. No `inf`. Just pure math.

## Features
- Numbers up to 1024 digits each
- Addition, subtraction, multiplication, division
- Power, factorial, multifactorial (`10 !!`, `15 !3`)
- GCD, LCM, integer square root
- Clean error messages (no crashes)

## How to Run

**Windows (Notepad + g++)**
1. Save `main.cpp` to a folder
2. Drag it onto `build.bat`
3. Done!
